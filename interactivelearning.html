    Ed Tech Tools

    Interactive Learning

    What is Interactive Learning?

    Interactive learning differs from traditional, passive learning in that it allows students to engage in hands-on learning that better imitates learning in the real world. There are many ed tech tools that allow students to interact with the content and other learners as they proceed through lessons. Tools that encourage interactive learning can be used during live instruction and during self-paced, independent learning.

    Interactive Learning Tools

    The following is a list of my favorite Ed Tech Tools used for interactive learning.

    Nearpod

    Nearpod has the potential to make learning interactive. It is a great tool to keep students engaged, formatively assess their understanding, and give them a voice.

    Visit Nearpod

    Edpuzzle

    Edpuzzle has a live mood and independent mood in which students watch videos to learn content. Teachers can insert notes, multiple choice questions, and open ended questions throughout the video so students can interact with the content.

    Visit Edpuzzle

    Google Jamboard

    Google Jamboard allows learners the opportunity to collaborate in real time, show their thinking, and express their thoughts.

    Visit Google Jamboard

    Seesaw

    Seesaw allows students to interact with content using the various tools within the tool. Students can also offer feedback and encouragement to their peers by interacting with each other on the feed.

    Visit Seesaw
